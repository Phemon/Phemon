【题解】UVA10918 – Tri Tiling

2019-02-14 OI 题解

题目描述传送门

题意：对3×n的面积使用1×2的块进行完美覆盖，计算方案总数。

1.若n为奇数，则3×n为奇数，所以用1×2的块无法进行完美覆盖，方案数为0。 2.若n为偶数，将这n列每2列分开，如果这一列与前面分开，就只有三种方法即 3×f[n-2]；若不分开，则不可分解的方案只有两种，即2×（f[n-4]+f[n-6]+……+f[0])。

由上可得 f[i]=3×f[i-2]+2×(f[i-4]+f[i-6]+…+f[0]) ① f[i-2]=3×f[i-4]+2×(f[i-6]+…f[0]) ② ①-②,得f[i]=4×f[i-2]-f[i-4].

#include<bits/stdc++.h>
using namespace std;
int main()
{
    int n;
    int a[31];
    a[0]=1;
    a[2]=3;
    int i;
    for(i=1;i<=30;i=i+2)
        a[i]=0;
    for(i=4;i<=30;i=i+2)
        a[i]=a[i-2]*4-a[i-4];
    while (scanf("%d",&n)&&n!=-1)
        printf("%d\n",a[n]);
    return 0;
}
